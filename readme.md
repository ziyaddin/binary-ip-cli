# binary-ip-cli [![Build Status](https://travis-ci.org/ziyaddin/binary-ip-cli.svg?branch=master)](https://travis-ci.org/ziyaddin/binary-ip-cli)

> Convert an IP address to binary


## Install

```
$ npm install --global binary-ip-cli
```


## Usage

```
$ binary-ip --help

  Usage
    $ binary-ip <ip> [...]

  Examples
    $ binary-ip <IP>
    <result>
```
